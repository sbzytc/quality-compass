# نظام رصدة - الوثيقة التفصيلية الكاملة
## Rasdah Quality Management System - Complete Features Document

---

## نظرة عامة | Overview

نظام رصدة هو منصة متكاملة لإدارة الجودة مصممة للشركات متعددة الفروع. يوفر النظام أدوات شاملة للتقييم والمراقبة والتحسين المستمر مع لوحات تحكم ذكية وتقارير تحليلية متقدمة.

Rasdah is a comprehensive quality management platform designed for multi-branch organizations. It provides complete tools for evaluation, monitoring, and continuous improvement with intelligent dashboards and advanced analytical reports.

---

## 1. نظام إدارة المستخدمين والصلاحيات | User & Role Management

### الأدوار المتاحة | Available Roles:
| الدور | Role | الوصف |
|-------|------|-------|
| مدير النظام | Admin | صلاحيات كاملة لإدارة جميع جوانب النظام |
| المدير التنفيذي | Executive (GM) | عرض شامل لأداء جميع الفروع والتقارير التنفيذية |
| مدير الفرع | Branch Manager | إدارة فرع محدد ومتابعة الملاحظات والإجراءات التصحيحية |
| المقيّم | Quality Auditor | إجراء التقييمات وتسجيل الملاحظات |
| موظف العمليات | Operations | تنفيذ المهام التشغيلية المسندة |
| موظف الفرع | Branch Employee | الوصول المحدود لتقديم طلبات الدعم |
| موظف الدعم الفني | Support Agent | إدارة تذاكر الدعم الفني |

### مميزات إدارة المستخدمين:
- ✅ إنشاء المستخدمين ودعوتهم عبر البريد الإلكتروني
- ✅ إدارة الصلاحيات بشكل مرن لكل دور
- ✅ طلبات تغيير البيانات الشخصية مع موافقة الإدارة
- ✅ تفعيل/تعطيل الحسابات
- ✅ إعادة تعيين كلمات المرور
- ✅ ربط المستخدمين بالفروع والمناطق

## 3. نظام التقييم الشامل | Comprehensive Evaluation System

### قوالب التقييم | Evaluation Templates:
- ✅ إنشاء قوالب تقييم مخصصة
- ✅ تحديد فترات التقييم (سنوي، شهري، أسبوعي)
- ✅ تقسيم التقييم إلى فئات (Categories) قابلة للتخصيص
- ✅ معايير تقييم (Criteria) لكل فئة مع أوزان ونقاط
- ✅ تحديد المعايير الحرجة (Critical Criteria)
- ✅ إصدارات متعددة للقوالب مع التحكم بالإصدار النشط

### عملية التقييم | Evaluation Process:
- ✅ إنشاء تقييم جديد واختيار الفرع والقالب
- ✅ حفظ التقييم كمسودة والعودة لإكماله لاحقاً
- ✅ تسجيل الدرجات لكل معيار (0-5 أو حسب القالب)
- ✅ إضافة ملاحظات نصية لكل معيار
- ✅ إرفاق صور ومستندات كأدلة
- ✅ حساب النسبة المئوية تلقائياً لكل فئة والتقييم الكلي
- ✅ تقديم التقييم للمراجعة والموافقة
- ✅ إمكانية التعديل خلال 24 ساعة بعد التقديم
- ✅ أرشفة التقييمات القديمة تلقائياً

### سجل التقييمات | Evaluation History:
- ✅ عرض جميع التقييمات السابقة
- ✅ فلترة حسب الفرع، الفترة، الحالة
- ✅ تصدير التقييمات
- ✅ أرشيف للتقييمات المكتملة

## 4. إدارة الملاحظات والمخالفات | Findings & Non-Conformities

### دورة حياة الملاحظة | Finding Lifecycle:
```
مفتوحة (Open) → قيد التنفيذ (In Progress) → بانتظار المراجعة (Pending Review) → محلولة (Resolved) / مرفوضة (Rejected)
```

### المميزات:
- ✅ إنشاء ملاحظات تلقائياً عند تسجيل درجات منخفضة
- ✅ إسناد الملاحظات لمدير الفرع المسؤول
- ✅ تحديد تاريخ استحقاق للحل
- ✅ رفع ملاحظات الحل مع المرفقات
- ✅ مراجعة الحلول من قبل المقيّم
- ✅ إمكانية رفض الحل مع توضيح السبب
- ✅ سجل كامل لجميع الإجراءات والتغييرات
- ✅ إشعارات تلقائية عند كل تغيير في الحالة
